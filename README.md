# Menyanthes (Freeware version)
Software for Time Series Analysis, originally developed by KWR and Delft University of Technology
(see a.o. https://www.sciencedirect.com/science/article/abs/pii/S1364815212001831)

----------------------------------- Installation and instructions (64-bit version) -------------------------------------

Installation steps:
1) Install MCRInstaller.exe if absent 	        	(Matlab Compiler Runtime, v8.3 / r2014a 64-bit)
2) Install vcredist_x64.exe if absent           	(Visual C Redistributables, 2012 64-bit)
3) Unzip Menyanthes vx.x (64 bit, dddd-mm-dd).zip       (Time Series Analysis Software, in empty directory)
